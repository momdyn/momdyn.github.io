Sensors
=======

Sensors measure translational or rotational motion of points or frames.

`Position Sensor <sensors/position_sensor.html>`_

A position sensor records the position of one point relative to a reference point, expressed in a reference frame.

.. raw:: html

   <hr>

`Velocity Sensor <sensors/velocity_sensor.html>`_

A velocity sensor records the velocity of a point expressed in a reference frame.

.. raw:: html

   <hr>

`Accelerometer <sensors/accelerometer.html>`_

An accelerometer records the acceleration of a point expressed in a reference frame.

.. raw:: html

   <hr>

`Rate Gyro <sensors/rate_gyro.html>`_

A rate_gyro records the angular velocity of a frame expressed in a reference frame.
